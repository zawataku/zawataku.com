雑記の使い方

2023-08-27

はじめまして．このホームページの管理人をしているざわたくです．
少し前に「雑記」のページを追加したのですが，特に書くこともないので今までずっと放置していました．
何も書かないというのもアレなので，これからは主に更新内容について詳しく書いていくページにしようかと思います．


それでは今回の更新についてです．
先述の通り雑記を書くことにしたため，雑記内ページのテンプレートっぽいものを作りました．
テンプレートとは言ったものの，（雰囲気で察せるとは思いますが）このホームページはhtml手打ちという昔ながらの更新方法なので， イマドキの「入力するだけで簡単！」みたいなものではありませんけどね...
需要があるかはわかりませんが，今回作ったテンプレートを記載しておきます．

↓template.htm（最後の部分はCSS書くのが面倒くさかったので<br>連打で...） 
<!DOCTYPE html>
<html lang="ja">
<meta charset="UTF-8">

<head>
    <title><!--タイトル--></title>
    <link href="/css/common.css" rel="stylesheet">
    <link href="/css/notes.css" rel="stylesheet">
</head>

<body background="/img/common/b056.jpg">
    <center>
        <div class="notes_title">
            <h1><!--タイトル--></h1>
            <h4><!--日付--></h4>
        </div>
    </center>
    <div class="notes_content">
        <!--本文-->
    </div>
    <center>
        <br>
        <a href="/notes.htm">「雑記」に戻る</a>
        <br>
        <br>
        <br>
    </center>
</body>

</html>

↓common.css 
@charset "UTF-8";

body {
    width: auto;
    height: 100%;
    overflow-x: hidden;
    -webkit-text-size-adjust: 100%;
}

↓notes.css 
@charset "UTF-8";

.notes_title {
    margin: 50px 0 50px 0;
}

.notes_content {
    margin: 10px 150px 10px 150px;
    padding: 10px;
    background-color: aliceblue;

}

.prettyprint ol.linenums>li {
    list-style-type: decimal;
}

td {
    padding: 15px;
}
